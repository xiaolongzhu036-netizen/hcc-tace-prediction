# 肝癌 TACE 疗效预测 · 智影肝策小组

基于 MRI 组学的肝癌 TACE 疗效预测系统（静态单页应用）。

## 本地预览

将仓库克隆后，用浏览器打开根目录下的 `index.html`，或使用任意本地静态服务器：

```bash
# Python 3
python -m http.server 8080
```

## GitHub Pages

1. 在本仓库 **Settings → Pages** 中，将 **Source** 设为 **Deploy from a branch**。
2. Branch 选择 `main`，文件夹 `/ (root)`，保存。
3. 数分钟后可通过 `https://xiaolongzhu036-netizen.github.io/<仓库名>/` 访问（请将 `<仓库名>` 替换为你的 GitHub 仓库名称）。

## 资源

- `assets/`：研究图表、预测报告 PDF 等静态资源。
